## **چرا Git؟**
توی پروژه های کوچیک که تعداد برنامه نویسیا کمه، معمولا هر برنامه نویسی روی یه قسمت از پروژه کار میکنه. وقتی هم بخوان یه تغییری بدن معمولا مشکلی ایجاد نمیشه چرا که یه نفر میره و تغییرات رو انجام میده.
